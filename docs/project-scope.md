# Project Scope

## Ziel

Das Zoo Management Tool bildet zentrale Verwaltungsprozesse eines Zoos als Uni-MVP ab. Es verwaltet Tiere, Arten, Gehege, Fuetterungsplaene, Gesundheits- und Pflegeeintraege, Aufgaben und Audit-Logs.

## Nutzergruppen

- Admin
- Keeper
- Vet
- Viewer

## MVP-Funktionen

- Login und Logout
- Serverseitige rollenbasierte Zugriffskontrolle
- Tierverwaltung mit Zuordnung zu Art und Gehege
- Arten- und Gehegeverwaltung
- Fuetterungsplaene
- Gesundheits- und Pflegeeintraege
- Aufgabenverwaltung
- Audit-Logging relevanter Aenderungen
- Dashboard mit Kennzahlen und Warnungen

## Nicht-Ziele

- Ticketverkauf, Zahlungsabwicklung oder Besuchertracking
- Biometrische Daten
- Echte personenbezogene Demo-Daten
- Externe Behoerdenschnittstellen
- KI-gestuetzte medizinische Empfehlungen
- Mobile App

## Definition of Done

- Backend und Frontend starten lokal reproduzierbar.
- Authentifizierung und RBAC funktionieren serverseitig.
- CRUD-nahe MVP-Funktionen sind ueber API und Frontend nutzbar.
- Audit-Logging ist fuer relevante Aenderungen vorhanden.
- Demo-Daten sind synthetisch.
- README, Datenschutz-/Security-Konzept, Threat Model, Compliance-Matrix und Testbericht liegen vor.
